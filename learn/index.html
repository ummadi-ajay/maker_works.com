Skills You Gain

Electronics & Circuit Design

Understand components, build circuits, and learn to prototype with breadboards and sensors.

Block-Based Coding

Create animations, games, and interactive stories using Scratch Jr, Scratch, and Micro:bit.

Mobile App Development

Design and build real Android apps using MIT App Inventor with drag-and-drop logic.

Robotics & Automation

Learn to build, program, and control robots for various tasks and competitive challenges.
